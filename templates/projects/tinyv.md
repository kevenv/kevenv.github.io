# TinyV

![]({{rootImages}}projects/tinyv-cover.jpg)

RISC-V emulator (tinyv)

A small RISC-V emulator.

- RV64I,M,Zicsr.
- Machine/User/Supervisor privilege levels.
- Exceptions.
- Basic C runtime.
- Console I/O.
- Passes some of the official tests suite.
