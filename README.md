# kevenv.github.io
Personal website

## Philosophy

- no web frameworks
- no useless js, jquery BS
- no complex packages management
- minimal css

## Install

`pip3 install --user -r tools/requirements.txt`

## Build

`python3 tools/build_site.py`

## Dependencies

- python3
- pystache
- markdown
- markdown-katex
- Pygments
- fontawesome.js

## Tools

- sublime text packages: mustache, markdown
